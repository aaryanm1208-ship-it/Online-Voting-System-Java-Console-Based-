# Online-Voting-System-Java-Console-Based-
A secure Online Voting System developed using Java that allows registered users to vote only once. The system maintains voter records, candidate details, and voting results using file storage.

Features
User (Voter) registration
Secure login using Voter ID
One vote per user (prevents duplicate voting)
Candidate list display
Vote counting
Result declaration
Data persistence using file handling

Technologies Used
Java
OOP Concepts
File Handling
Collections Framework
